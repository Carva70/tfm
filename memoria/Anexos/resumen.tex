\chapter*{Abstract}

Este TFM tiene como objetivo estudiar las distintas vulnerabilidades relacionadas con el prompt injection en un contexto corporativo.
Se simularán entornos con modelos de lenguajes locales (LLaMA, etc.) para reproducir escenarios de ataque. 

Incluirá la propuesta de un conjunto de controles de mitigación en contextos reales.
